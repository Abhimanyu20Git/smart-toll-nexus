# SmartToll Connect

Highway toll management system with role based dashboards for users, admins, and operators. Built with HTML, CSS, and JavaScript.

## Features

- Landing page with hero section and feature cards
- Role based authentication (User, Admin, Operator)
- Login and registration with simulated OTP verification
- User dashboard with wallet balance and transaction history
- Wallet recharge with preset amount options
- Admin dashboard with revenue stats and toll booth management
- Add, edit, and delete toll booths
- Operator dashboard with current assignment info
- Live vehicle monitoring with real time status updates
- Dark mode across all pages
- Fully responsive layout
- Toast notifications

## Tech Stack

- HTML5
- CSS3 (Custom Properties, Flexbox, Grid)
- Vanilla JavaScript
- localStorage for session management

## Pages

| Page | Description |
|------|-------------|
| index.html | Landing page with hero, features, and CTA |
| auth.html | Login and registration with OTP flow |
| user-dashboard.html | Wallet, vehicle info, transactions |
| admin-dashboard.html | Stats, toll booth CRUD |
| operator-dashboard.html | Assignment, live vehicle feed |

## Getting Started

Open `index.html` in a browser. No build tools needed.
